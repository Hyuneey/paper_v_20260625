# Multi-Rule Outer Validation Protocol

Stage E2X-V begins only from a clean commit containing the inner panel and arm freeze. All 100 panel rules run twice on their own KPI outer values. Outer labels are unavailable to containers and are loaded only after every individual prediction and frozen OR composition has been hash-recorded privately.

Only the four committed arm-level predictions receive outer metrics. Individual outer-rule metrics are neither computed nor exposed to selection. The primary comparison is Coverage-3 OR versus Best-1. KPI-weighted macro, pooled micro, ten-KPI distributions, and descriptive paired bootstrap intervals are reported. The held-out test remains sealed.
