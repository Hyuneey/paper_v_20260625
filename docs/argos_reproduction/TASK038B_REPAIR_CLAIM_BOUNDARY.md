# TASK-038B Repair Claim Boundary

TASK-038B asks whether one bounded RepairAgent revision recovers isolated
runtime operability for the thirteen frozen TASK-037D failures.

It may report:

- reproducible failure count;
- provider response and token usage;
- extraction and static-validity yield;
- deterministic target/contrast runtime recovery;
- structural semantic-drift diagnostics;
- A1/A3 branch-state updates;
- primary and conditional Repair recovery rates.

It does not evaluate ReviewAgent, inner or outer detection performance,
detector-rule fusion, the Full Aggregator, outer generalization, or sealed-test
performance. `passed_repair_agent_operability_experiment` means only that the
bounded experiment completed correctly; effectiveness is classified
separately.
