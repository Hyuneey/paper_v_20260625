# E5/E6 Outer Exposure Boundary

TASK-037E is a paper-aligned follow-up outer-validation experiment. It is not
an untouched confirmatory experiment and is not exact ARGOS reproduction.

The TASK-037E outer partition is a previously exposed follow-up validation
partition. Rule generation used only the generation partition and rule
selection used only the inner partition, but the broader experiment design
followed prior inspection of outer results. Therefore TASK-037E does not
support an untouched confirmatory superiority claim.

TASK-035B, TASK-037B, and TASK-037C previously reported results on the same
outer partition. TASK-037E therefore reports all predeclared arms and
contribution costs, retains both LSTMAD variants, and makes only descriptive
comparisons with inherited generic-rule results.

The sealed test remains unauthorized. No final superiority, benchmark,
variant-winner, or thesis headline claim is permitted.
