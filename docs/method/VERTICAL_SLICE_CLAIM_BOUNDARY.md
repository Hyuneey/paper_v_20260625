# Vertical-Slice Claim Boundary

## Allowed claim

The complete delayed-response contract pipeline is connected and
deterministically replayable on predeclared synthetic fixtures.

## Not established

- real dataset execution;
- learned graph quality;
- rule-generation quality;
- calibration validity;
- anomaly-detection performance;
- explanation usefulness;
- physical causality or root cause;
- detector or fusion behavior;
- method completion or thesis completion.

The report contains hashes, status values, reference identifiers, and bounded
scenario outcomes only. It contains no source/target arrays, dataset rows,
provider material, generated Python, detector outputs, or fusion outputs.
